# **App Name**: Aisha & Zaid: Grand Nuptials

## Core Features:

- Cinematic Page Introduction: A full-screen cinematic intro with a slowly drawing golden mandala SVG and deep parallax hero section, creating an immersive entry experience.
- AI-Generated Wedding Storyteller Tool: A tool to generate elegant, romantic placeholder copy for 'Our Story' section about the couple's journey, aligning with the premium editorial tone.
- Dynamic Countdown & Event Timeline: An interactive live JavaScript countdown timer with 3D card-flip animations, and an event timeline with drawing line animations on scroll.
- Luxurious RSVP System: An elegantly styled RSVP form with animated input field glows on focus, and a golden confetti explosion on (fake) successful submission.
- Curated Editorial Content Display: Sections for 'Our Story' (with two-column layout and asymmetric photo styling), 'Photo Gallery' (with masonry grid and hover effects), and 'Venue & Map' details with embedded map.
- Animated Scroll & Reveal Engine: Staggered clip-path scroll reveal animations for all section elements, subtle gold shimmer on section dividers, and smooth scroll behavior.
- Atmospheric Background & Detailing: Floating golden rose petal SVGs, subtle Islamic geometric pattern watermarks, dark image vignette overlays, and an animated frosted glass navbar.

## Style Guidelines:

- Primary background: Deep midnight navy (#0a0d1a) to establish an opulent, dark mode foundation, evocative of twilight and royalty.
- Text & soft accents: Warm ivory (#f9f4ec) providing high contrast for legibility while maintaining a soft, inviting feel against the dark base.
- Key accent: Antique gold (#c9a84c) used for prominent details, interactive elements, and intricate ornamental graphics, symbolizing luxury and warmth.
- Secondary accent: Dusty rose (#c2847a) applied sparingly to introduce a gentle, romantic warmth and subtle visual depth.
- Headline and names font: 'Cormorant Garamond' (serif) for an elegant, high-contrast editorial feel. Note: currently only Google Fonts are supported.
- Body and details font: 'Jost' (sans-serif) for clean, geometric readability that complements the display font. Note: currently only Google Fonts are supported.
- Utilize bespoke gold SVG elements such as an Arabic Bismillah, a blooming mandala, custom wedding-themed icons (ring, camera, etc.), ornamental dividers, and an Islamic arch frame, all integrated seamlessly for luxury.
- A fluid mix of full-viewport hero, two-column content sections, a dynamic masonry photo grid, and a responsive vertical/horizontal timeline, all within a max content width of 1100px and generous clamp(80px, 8vw, 140px) vertical padding.
- Layered surfaces feature deep navy backgrounds with lighter navy cards and translucent frosted glass panels for an immersive, tactile depth, adapting to single-column on mobile.
- A diverse set of cinematic animations including a page-load mandala draw, hero parallax, floating golden rose petals, letter-by-letter couple name reveal, card-flip countdown numbers, and CSS gold shimmer sweeps on dividers.
- Interactive elements benefit from scroll reveals using staggered clip-path wipes, input border glows on focus, confetti burst on RSVP submit, dynamic navbar transparency with blur on scroll, and detailed photo gallery hover effects (zoom, gradient, caption slide).
